# Legal PDFs (static files)

Privacy policy PDFs live here. **Use ASCII filenames only** so production hosts/CDNs can serve them reliably (non-ASCII paths often break after URL encoding).

| File | Language |
| ---- | -------- |
| `privacy-en.pdf` | English (`en`) |
| `privacy-ru.pdf` | Russian (`ru`) |
| `privacy-hy.pdf` | Armenian (`hy`) |

The app maps the current UI language to these files (footer **Privacy Policy** and checkout).

Served from the site root, e.g. `/legal/privacy-hy.pdf`.
