# Legal PDFs (static files)

Privacy policy PDFs live here. **Filenames must match exactly** (including spaces and script):

| File | Language |
| ---- | -------- |
| `Privacy Policy.pdf` | English (`en`) |
| `Политика конфиденциальности.pdf` | Russian (`ru`) |
| `Գաղտնիության քաղաքականություն.pdf` | Armenian (`hy`) |

The app maps the current UI language to these files (footer **Privacy Policy** and checkout). URLs are encoded automatically.

Served from the site root, e.g. `/legal/Privacy%20Policy.pdf`.
